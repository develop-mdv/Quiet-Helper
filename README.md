# Quiet Helper

Приватный десктоп-ассистент для Windows со стелс-оверлеем. Помогает отвечать на вопросы
с экрана и из разговора и **не виден при демонстрации экрана** (Zoom / Teams / OBS / Discord).
Свой аналог Cluely / OpenCluely / Pluely.

> ⚠️ **Ответственное использование.** Слушание разговора может подпадать под законы о
> согласии на запись. Стелс-режим скрывает окно от программного захвата экрана, но не от
> физической камеры. Использование на собеседованиях / экзаменах может нарушать их правила —
> это ваша ответственность.

## Возможности

- **Вопрос с экрана** — горячей клавишей выделяете область (или весь экран) → скриншот уходит
  в мультимодальную модель → ответ стримится в оверлей (с рендером Markdown и **формул KaTeX**).
- **Слушание разговора** — захват микрофона и системного звука → распознавание речи → детекция
  вопросов → ответ (ручной или автоматический).
- **Умная детекция вопросов** — эвристика (вопрослова, «ли», императивы) + опциональный
  LLM-классификатор «вопрос ко мне?», чтобы авто-ответ не срабатывал на болтовню.
- **История ответов** — новые ответы копятся, между ними можно листать (◀/▶); просмотр старого
  ответа не прерывается новым (он формируется в фоне).
- **База знаний (RAG)** — прикрепляете текст/файлы (.txt/.md/.pdf), и ответы ищутся в них:
  лексический (BM25), **семантический** (эмбеддинги) или **гибридный** поиск. Нет ответа в
  материале → обычный ответ.
- **Стелс-оверлей** — `setContentProtection` (Windows `WDA_EXCLUDEFROMCAPTURE`), always-on-top,
  вне таскбара, клик-сквозь, паника-скрытие, изменяемый размер.
- **Гибкий доступ к LLM**:
  - **API-ключ** — [AllTokens](https://alltokens.ru) (OpenAI-совместимый), по умолчанию
    `google/gemini-3.1-flash-lite`.
  - **Подписка ChatGPT через Codex** — ответы (`gpt-5.6-sol`) и даже транскрибация (`whisper-1`)
    по вашей подписке, без отдельного API-ключа.
  - OAuth Google / Anthropic — заготовки на будущее.

## Стек

Electron + electron-vite + React + TypeScript. `openai` SDK (chat/vision/streaming). Локальный
Whisper (whisper.cpp), облачный STT AllTokens или встроенный ASR Codex. `pdf-parse` для PDF,
KaTeX для формул.

## Запуск (dev)

```bash
npm install
npm run dev
```

При первом запуске откроются настройки: выберите провайдера, введите ключ AllTokens (или войдите
через Codex), нажмите «Проверить», затем «Завершить настройку».

## Сборка

```bash
npm run build      # собрать в out/
npm run start      # запустить собранное
npm run dist       # установщик (electron-builder)
```

## Горячие клавиши (по умолчанию, настраиваются)

| Действие | Клавиши |
| --- | --- |
| Показать / скрыть оверлей | `Ctrl` + `\` |
| Выделить область → вопрос | `Ctrl` + `Shift` + `S` |
| Скриншот экрана → вопрос | `Ctrl` + `Shift` + `A` |
| Ответ по последнему вопросу из разговора | `Ctrl` + `Shift` + `Space` |
| Слушание вкл / выкл | `Ctrl` + `Shift` + `L` |
| Паника (мгновенно скрыть) | `Ctrl` + `Shift` + `X` |
| Сквозной режим (для ввода текста) | `Ctrl` + `Shift` + `C` |
| Предыдущий / следующий ответ | `Ctrl` + `Alt` + `←` / `→` |
| Двигать оверлей | `Ctrl` + стрелки |
| Скролл ответа | `Alt` + `↑` / `↓` |

## Модели

- **AllTokens (по ключу):** `google/gemini-3.1-flash-lite` (дёшево, фото+аудио, по умолчанию),
  `google/gemini-3.5-flash`, `anthropic/claude-sonnet-5`, `x-ai/grok-4.5`, `qwen/qwen3.7-plus`.
- **Codex (по подписке ChatGPT):** `gpt-5.6-sol` (флагман, по умолчанию), `gpt-5.6-terra`
  (дешевле). Голый алиас `gpt-5.6` и `gpt-5.6-luna` бэкендом Codex не принимаются.

## Распознавание речи (STT)

Три режима на выбор (Настройки → «Слушание разговора»):

1. **Локальный Whisper** — офлайн, бесплатно, приватно. Бинарник и модель в репозиторий не входят:
   - положите `whisper-cli.exe` в `resources/whisper/` и модель `ggml-<name>.bin` в
     `resources/whisper-models/` ([whisper.cpp](https://github.com/ggerganov/whisper.cpp));
   - либо задайте пути через `WHISPER_BIN` и `WHISPER_MODEL`.
2. **Codex (подписка ChatGPT)** — `whisper-1` через `chatgpt.com/backend-api/transcribe`, по вашей
   подписке (нужен `codex login`).
3. **Облако AllTokens** — аудио как `input_audio` в мультимодальную Gemini (нужен ключ AllTokens).

Если выбран локальный Whisper, но он не установлен — авто-фолбэк на облако AllTokens.

## База знаний (RAG)

Настройки → «База знаний»: добавляйте текст или файлы (.txt / .md / .pdf), редактируйте, удаляйте.
Ответы сначала ищутся в материалах; если ответа там нет — модель отвечает как обычно.

- **Тип поиска:** лексический (BM25, офлайн), семантический (эмбеддинги) или гибридный (по умолчанию).
- **Эмбеддинги** идут по ключу **AllTokens** (модель `google/gemini-embedding-2` по умолчанию;
  также `baai/bge-m3`, `openai/text-embedding-3-*`, `qwen/qwen3-embedding-8b`) — **независимо** от
  провайдера ответов, поэтому семантика работает и при ответах через Codex (ключ вводится прямо в
  разделе базы). Векторы считаются один раз и кешируются.
- Небольшая база (< 20k символов) подаётся модели целиком; большая — по релевантным кускам.
- Для скриншот-вопросов по большой базе вопрос сначала извлекается с картинки vision-запросом.
- Быстрый тумблер базы — кнопка 📚 в шапке оверлея.

## Провайдер ChatGPT через Codex

Установите [Codex CLI](https://developers.openai.com/codex/cli), выполните `codex login` — токен
из `~/.codex/auth.json` используется автоматически. Ответы идут на `chatgpt.com/backend-api/codex`,
транскрибация — на `chatgpt.com/backend-api/transcribe`, всё списывается с подписки ChatGPT. Путь
неофициальный для сторонних приложений и может перестать работать при изменениях OpenAI. Только
личный аккаунт, без пула/раздачи токенов.

## Структура проекта

```
src/
  main/
    knowledge/   # база знаний: хранилище, чанкинг, BM25, эмбеддинги, PDF
    llm/         # провайдеры: AllTokens (OpenAI-совместимый), OAuth (Codex/Google/Anthropic)
    stt/         # распознавание речи: whisper / codex / cloud
    ...          # окна, стелс, хоткеи, захват экрана, детекция/классификация вопросов, IPC
  preload/       # безопасный мост contextBridge
  renderer/      # React: overlay / settings / selection
  shared/        # общие типы и имена IPC-каналов
resources/       # иконки (whisper-бинарник и модели — вне репозитория)
```
